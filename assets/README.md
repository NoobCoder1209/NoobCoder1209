# assets/

| File | Source | Refresh cadence |
|------|--------|-----------------|
| `metrics.svg` | `lowlighter/metrics` (GitHub Action) | Daily 04:37 UTC |
| `project-card-*.svg` | Hand-designed, committed by hand | Whenever featured projects change |

Snake animation lives on the `output` branch (not here), maintained by the same daily Action.
